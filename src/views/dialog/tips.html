消息提示弹框

消息提示弹框主要用于提示内容较少的纯文本的弹框
知识点：内容同样可以html标签
案例1：普通提示框
知识点：最普通的提示
代码如下：
<a class="btn btn-iroad mt" href="javascript:$.ligerDialog.alert('普通提示')" >普通提示</a>

普通提示

<a class="btn btn-iroad mt" href="javascript:uestionBtn.question()" >询问提示</a> <script type="text/javascript"> var questionBtn={ dialogBtn :[ { text: '确定', onclick: function (dialog,settings) { alert(settings.id); alert("确定删除"); return false; } }, { text: '取消', onclick: null } ], settings:{ id: 123 }, question:function () { $.ligerDialog.question("您确定要删除该记录吗?",this.dialogBtn,this.settings); } } </script>

案例8：通用提示框
知识点：通用提示框可以提示alert success warn info error question等几种类型的提示，可通过 $.ligerDialog.openCDialog()函数来调用
代码如下：
<a class="btn btn-iroad mt" href="javascript:showCDialog()" >询问提示</a> <script type="text/javascript"> function showCDialog(){ var dialogBtn = [ { text: '确定', onclick: function (dialog,settings) { alert(settings.id); } }, { text: '取消', class:'cancel', onclick: null } ]; var settings={ id: 123 } var type="question"; //alert success warn info error question var content="请问您是否要删除这条记录?"; /** * type:表示提示的类型 alert success warn info error question * content:表示提示的内容 * dialogBtn：提示框按钮（可选） * settings：参数（可选） */ $.ligerDialog.openCDialog(type,content,dialogBtn,settings); } </script>
效果如下：
询问提示
案例9：自定义提示框
知识点：自定义提示框同样可以提示alert success warn info error question等几种类型的提示， 并且还可以自定义弹框的一些参数，比如弹框高度宽度等。可通过$.ligerDialog.openTDialog()函数来调用
代码如下：
<a class="btn btn-iroad mt" href="javascript:openTDialog()" >自定义提示框</a> <script type="text/javascript"> function openTDialog(){ var dialogBtn = [ { text: '确定', onclick: function (dialog,settings) { alert(settings.id); } }, { text: '取消', class:'cancel', onclick: null } ]; var settings={ id: 123 } var type="question"; //alert success warn info error question var title="请您确认"; var content="请问您是否要删除这条记录?"; var dialogCfn={ type:type, title:title, content:content, buttons:dialogBtn, settings:settings } /** * 生成弹框 * 作者：姚明 * 时间：2017-11-17 * @param dialogCfn * type : "alert" ,提示框类型(必填) 可取值为：alert,success,warn,info,error,question * title : "默认标题", 弹框标题（可选） * titleClass : "标题的class类"（可选） * content : "提示内容",提示内容（可选） * width : 300, 弹框宽度（可选，默认300px） * 备注： 0<width<=1 代表弹框宽度百分比 width>1 代表弹框宽度的实际像素 * height : auto, 弹框内容高度（可选，默认根据内容自适应） * 备注： 0<height<=1 代表弹框高度百分比 height>1代表弹框内容高度的实际像素 * buttons : dialogBtn, 弹框按钮 （可选） * settings : { 弹框自定义参数 （可选） * method : "add", * id:"123456" * } */ $.ligerDialog.openTDialog(dialogCfn); } </script>
效果如下：
自定义提示框
