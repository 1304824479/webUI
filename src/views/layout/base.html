边框布局

边框布局分为左右边框和上下左右中边框两种
知识点：无
案例1：左右边框
知识点：左右边框布局，需要class为layout的父元素与属性为position="left" 及position="center"子元素的配合使用。同时需要给父元素绑定initLayout()函数进行初始化，并且需要传递一个参数。 其中，layout的宽高默认与父元素的宽高一致
代码如下：
html代码 <div class="layout"> <div position="left">左边布局</div> <div position="center">中间布局</div> </div> js代码 <script type="text/javascript"> $(function () { var layoutCfg={leftWidth :230}; //边框左边的宽度为230像素 $(".layout").initLayout(layoutCfg); //对左右边框进行初始化 }) </script>

案例2：上下左右中边框
知识点：上下左右中边框布局，需要class为layout的父元素与属性为position="top" ，position="left"，position="center"，position="right"，position="bottom"子元素的配合使用。 同时需要给父元素绑定initLayout()函数进行初始化，并且需要传递一个参数。 其中，layout的宽高默认与父元素的宽高一致
代码如下：
html代码 <div class="layout"> <div position="top">上部布局</div> <div position="left">左边布局</div> <div position="center">中间布局</div> <div position="right">右边布局</div> <div position="bottom">底部布局</div> </div> js代码 <script type="text/javascript"> $(function () { var layoutCfg={ topHeight :150, //顶部高度 leftWidth :230, //左边宽度 rightWidth:150, //右边宽度 bottomHeight:100 //底部高度 }; $(".layout").initLayout(layoutCfg); //对左右边框进行初始化 }) </script>
效果如下（示例的高度为600px）：
