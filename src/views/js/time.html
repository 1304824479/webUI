时间控件分为日期控件和日期时间控件两种

知识点：日期控件只需要在input标签中加上iroad-date即可
代码如下：
<input type="text" class="iroad-date" > $(function(){ cfg:可选项 cfg.skin:皮肤颜色 可选数字0到9，或者直接是颜色值 var cfg={ skin:0 } calendar.init(cfg); 举例：calendar.init({skin:9}); });
