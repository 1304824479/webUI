<select name="select2" data-url="http://localhost:7878/city" > <option value="">请选择</option> <option value="11">天门</option> <option value="22">武汉</option> <option value="33">厦门</option> <option value="44">仙桃</option> </select> <script type="text/javascript"> $(function () { $("select[name='select2']").selectOnChange(); }) </script>

<select name="select4" data-init-url="http://localhost:7878/city" ></select> <script type="text/javascript"> $(function () { $("select[name='select4']").selectInit(function(obj){ $(obj).val(22); }); }) </script>

案例6：初始化下拉选项框并赋值且联动
知识点：初始化下拉选项框需要有一个data-init-url属性并且联动下拉选项框需要有一个data-url属性
代码如下：
<select name="select6" data-init-url="http://localhost:7878/city" data-url="http://localhost:7878/city" data-value="22,44,11" ></select> <script type="text/javascript"> $(function () { $("select[name='select6']").selectInit(function (obj) { var value=$(obj).attr("data-value"); if(value&&value!=""){ var arr=value.split(","); var length=arr.length; var firstValue=arr[0]; $(obj).val(firstValue); if(length>1){ arr.remove(0); $(obj).loadSelectInitValue(firstValue,arr); } } }); }) </script>
