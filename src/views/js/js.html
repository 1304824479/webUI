js函数调用

系统中自定义了js的相关函数，页面中可以直接调用
知识点：无
案例1：数据校验
知识点：无
代码如下：
/** * 验证是否合法 * @param reg 正则表达式 * @param str 需要校验的字符串 * @returns {boolean} */ var result=verific.isLegal(reg,str); /** * 验证是否为数字 * @param str 需要校验的字符串 * @returns {boolean} */ var result=verific.isDigit(str); /** * 验证是否为空 * @param str * @returns {boolean} */ var result=verific.isEmpty(str);
