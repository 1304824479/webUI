按钮

系统的按钮样式
知识点：使用不同的按钮样式来控制按钮
案例1：不同状态图标按钮，支持a,input标签
知识点：要用a标签实现按钮图标只需要在a标签上加上class为btn即可实现a标签通用按钮图标, 并且提供除系统之外的两种不同颜色的按钮灰色（btn-gray）和蓝色（btn-blue）灰白色（btn-pale）
代码如下：
<a class="btn" href="#" >确定</a> <a class="btn cancel" href="#" >取消</a> <a class="btn disabled" href="#" >禁用</a> <input type="button" value="确定" class="btn" > <input type="button" value="取消" class="btn cancel" > <input type="button" value="禁用" class="btn disabled" > <input type="submit" value="确定" class="btn" > <input type="submit" value="取消" class="btn cancel" > <input type="submit" value="禁用" class="btn disabled" > <a class="btn" href="#" >默认颜色</a> <a class="btn btn-gray" href="#" >灰色</a> <a class="btn btn-blue" href="#" >蓝色</a> <a class="btn btn-pale color-black" href="#" >灰白色</a>
效果如下：
确定 取消 禁用
默认颜色 灰色 蓝色 灰白色
案例2：带有图标的按钮
知识点：带图标的按钮仅支持与a标签一起使用
代码如下：
<a class="btn" href="#" ><i class="icon iconfont icon-refresh " ></i> 刷新</a> <a class="btn" href="#" ><i class="icon iconfont icon-add "></i> 添加</a> <a class="btn" href="#" ><i class="icon iconfont icon-shitu_ "></i> 更多流程</a>
效果如下：
刷新 添加 更多流程
案例3：链接图标按钮
知识点：连接图标按钮仅支持a标签使用

添加 修改 查看 删除
